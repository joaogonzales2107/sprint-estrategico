# sprint-estrategico



Recapitulando o que você tem agora: um app instalável na tela inicial com ícone próprio, login, plano de 4 semanas com fichas de leitura narradas em áudio, lição do dia que segue o pilar em andamento, lembrete no navegador e alarme diário às 8h no celular. Do link de um anúncio no Instagram até aqui, você acabou construindo uma ferramenta melhor que o produto que te ofereceram.

Agora vem a parte que nenhuma ferramenta faz por você: segunda-feira às 8h o alarme toca, e a primeira tarefa é escrever a pergunta central de um problema real antes de discutir solução. A entrega de sexta — a issue tree usada numa reunião de verdade — é o que separa este sprint de mais um curso começado e abandonado.
